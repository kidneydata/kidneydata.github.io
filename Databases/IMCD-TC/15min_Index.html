Time Course of Phosphorylation Changes in Response to Vasopressin in Rat Renal Inner Medullary Collecting Duct

This table lists phosphorylation sites found to change in abundance following a step increase in vasopressin analog dDAVP (1 nM) in native rat inner medullary collecting duct. Collecting ducts were isolated from rat kidneys and studied in suspension. Quantification was by the TMT method using LC-MS/MS. Each time point was compared with an independent vehicle time-control. The amino acid sequences are centralized with respect to the phosphorylated amino acid.

The data for total protein abundance can be downloaded here. The full list of quantified phosphopeptides can be downloaded here.
